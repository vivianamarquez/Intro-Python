# Intro-Python

A beginner-friendly introduction to Python, built as a set of Jupyter notebooks for classroom use. Each lesson introduces new concepts with explanations and live coding examples; exercises give students standalone problems to practice on their own.

## What's inside

| Folder | Who it's for | Description |
|--------|--------------|-------------|
| [`Lessons Complete/`](Lessons%20Complete/) | Instructors | Full lessons with all code cells filled in and ready to run |
| [`Lessons Blank/`](Lessons%20Blank/) | Students | Same lessons, but code cells are empty — students type along during class |
| [`Exercises/`](Exercises/) | Students | Practice problems with instructions and blank code cells |
| [`Exercises Completed/`](Exercises%20Completed/) | Instructors | Answer keys for the exercise notebooks |

## Curriculum

Work through the notebooks in order:

| # | Lesson | Topics |
|---|--------|--------|
| 01 | [Introduction to Python](Lessons%20Complete/01_Introduction_to_Python.ipynb) | `print`, variables, data types, arithmetic, comparisons, logical operators |
| 02 | [Data Structures](Lessons%20Complete/02_Python_Data_Structures.ipynb) | Text strings, lists, tuples, sets, dictionaries |
| 03 | [Conditionals and Loops](Lessons%20Complete/03_Python_Conditionals_and_Loops.ipynb) | `if` / `elif` / `else`, `for` and `while` loops, list comprehension |
| 04 | [Functions and Libraries](Lessons%20Complete/04_Python_Functions_and_Libraries.ipynb) | Writing functions, `random`, `datetime`, NumPy |

Each lesson in `Exercises/` has a matching practice notebook with 7 short problems that apply the same ideas in new scenarios.


### Running the notebooks

Clone the repository, then start Jupyter from the project folder:

```bash
git clone <repository-url>
cd Intro-Python
jupyter notebook
```

Open notebooks from the folder that matches your role — `Lessons Blank` or `Exercises` for students, `Lessons Complete` or `Exercises Completed` for instructors.
